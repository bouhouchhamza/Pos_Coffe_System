# Installer Bimik_Cafe en PWA

## PC Chrome ou Edge

1. Ouvrez l'URL de production de Bimik_Cafe.
2. Cliquez sur l'icône d'installation dans la barre d'adresse.
3. Sinon, ouvrez le menu du navigateur puis choisissez **Installer Bimik_Cafe**.

## Android Chrome

1. Ouvrez l'URL de production.
2. Ouvrez le menu Chrome.
3. Choisissez **Ajouter à l'écran d'accueil** ou **Installer l'application**.

## iPhone Safari

1. Ouvrez l'URL de production dans Safari.
2. Appuyez sur le bouton de partage.
3. Choisissez **Ajouter à l'écran d'accueil**.

## Notes importantes

- L'installation PWA nécessite HTTPS en production.
- `localhost` fonctionne pour les tests de développement.
- Pour tester sur téléphone, déployez l'application sur une URL HTTPS ou utilisez une configuration locale HTTPS correcte.
